    2D Plots¶

    2D plots show telemetry channels as time-series or lap-distance traces. They appear in the lower panel after running the graph.

    Variable Tree¶

    The left side of each 2D plot has a variable tree with two sections:

    • Segments (top) — toggle individual stints or laps on/off
    • Variables (bottom) — toggle telemetry channels on/off

    Each variable can be expanded to show sub-toggles for different data types (Raw, Pred, Val).

    Interacting with Plots¶

    Action Control
    Pan Click and drag
    Zoom Scroll wheel
    Zoom region Right-click and drag
    Cursor Hover to see values
    Auto-range Right-click > "View All"

    Duplicating Plots¶

    Right-click a tab and select Duplicate tab to new panel (below) to create a second 2D plot panel. This lets you view different channel groupings side by side — for example, tire temps in one panel and driver inputs in another.

    Multiple Runs¶

    When you run the graph multiple times with different data, each run creates a new tab in the 2D plot panel. Use the tab bar to switch between results.
