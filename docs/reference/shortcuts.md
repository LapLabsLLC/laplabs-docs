# Keyboard Shortcuts

## 3D Track View

| Action | Control |
|--------|---------|
| Move camera in/out | Scroll wheel |
| Zoom (FOV) | Ctrl + scroll wheel |
| Rotate camera | Right-click + drag |
| Pan | Middle-click + drag |
| Draw with pen | Left-click + drag |

### Free Camera

| Action | Control |
|--------|---------|
| Move forward/back | W / S |
| Strafe left/right | A / D |
| Move up | Space |
| Move down | Alt |
| Sprint | Shift |
| Look around | Right-click + drag |

### Drone Camera (Gamepad)

| Action | Control |
|--------|---------|
| Throttle / Yaw | Left stick |
| Pitch / Roll | Right stick |
| Camera tilt up | LB |
| Camera tilt down | RB |

## 2D Plot

| Action | Control |
|--------|---------|
| Pan | Left-click + drag |
| Zoom region | Right-click + drag |
| Zoom | Scroll wheel |
| Cursor values | Hover |

## Console

| Shortcut | Action |
|----------|--------|
| `Ctrl+A` | Select all |
| `Ctrl+C` | Copy |
