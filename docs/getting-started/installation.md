# Installation

## Requirements

- **Windows 10/11** (64-bit)
- **iRacing** (for telemetry data — required to generate `.ibt` files)

## Download

Download the latest release from your distribution link. You'll receive a zip file containing the application.

## Setup

1. **Extract** the zip to a folder of your choice (e.g., `C:\LapLabsStudio`)
2. **Run** `LapLabsApp_Lite.exe`
3. On first launch, the app automatically creates a data folder with default configuration, tracks, vehicles, and dash layouts

!!! note
    No installer required — the app is fully portable. To uninstall, simply delete the extracted folder and the `.laplabsstudio` folder in your user directory.
