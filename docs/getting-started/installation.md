# Installation

## Requirements

- **Windows 10/11** (64-bit)
- **iRacing** (for live telemetry features — not required for offline analysis)

## Download

Download the latest release from your distribution link. You'll receive a zip file containing the application.

## Setup

1. **Extract** the zip to a folder of your choice (e.g., `C:\LapLabsStudio`)
2. **Run** `LapLabsApp.exe` (full version) or `LapLabsApp_Lite.exe` (lite version)
3. On first launch, the app automatically creates `~/.laplabsstudio/` with default configuration, tracks, vehicles, and dash layouts

!!! note
    No installer required — the app is fully portable. To uninstall, simply delete the extracted folder and `~/.laplabsstudio/`.
